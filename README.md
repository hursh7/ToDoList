### About the Project  
* 노마드코더의 Vanilla JS 실습용 프로젝트. 
* ```OpenWeatherAPI``` 를 통해 현재 위치 기반 날씨 구현. (현재는 텍스트로 구현, 추후에 아이콘 추가 예정.)
